# Trading Data

### File format  : exchange-currency-timeframe.csv
e.g. bitfinex-BTC-USDT-1d.csv 

#### Timeframe intro
- 1d : 1 day
- 1h : 1 hour
- 30m: 30 minutes
- 15m: 15minutes
- 5m : 5 minutes
- 1m : 1 minute
